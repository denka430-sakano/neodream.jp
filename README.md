## Formspree（フォーム）設定

1. https://formspree.io/ で無料アカウント作成
2. 「New Form」→ フォームIDを取得（例: `xyzabcde`）
3. `index.html` の下記を書き換え:

```html
<!-- 変更前 -->
<form ... action="https://formspree.io/f/YOUR_FORM_ID" ...>

<!-- 変更後 -->
<form ... action="https://formspree.io/f/xyzabcde" ...>
```

## GitHub Pages 公開手順

```bash
# 1. フォルダに移動
cd neodream-v3

# 2. Gitを初期化
git init
git add .
git commit -m "Initial commit: NeoDream website v3"

# 3. GitHubにリポジトリ作成後
git branch -M main
git remote add origin https://github.com/あなたのユーザー名/neodream.git
git push -u origin main

# 4. GitHub → Settings → Pages → Branch: main → Save
# → https://あなたのユーザー名.github.io/neodream/ で公開
```

## 独自ドメイン（neodream.jp）設定

`neodream-v3/` フォルダに `CNAME` ファイルを作成し、内容を以下に:

```

```
